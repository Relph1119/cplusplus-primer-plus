# 习题2.1

&emsp;&emsp;编写一个C++程序，它显示您的姓名和地址。

**解答：**  
代码位置：`exercises/ch02/ex01.cpp`
```c++
#include "iostream"
using namespace std;

int main() {
    cout << "Name: Hu Relph" << endl;
    cout << "Address: NanJing City JiangSu Province" << endl;
    return 0;
}
```
