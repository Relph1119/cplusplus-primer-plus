# 习题5.3

&emsp;&emsp;编写一个要求用户输入数字的程序。每次输入后，程序都将报告到目前为止，所有输入的累计和。当用户输入0时，程序结束。

**解答：**  
代码位置：`exercises/ch05/ex03.cpp`
```c++
#include "iostream"

using namespace std;

int main() {
    double number, sum;
    sum = 0;

    cout << "Please input a number:";
    while (cin >> number && number != 0) {
        sum += number;
        cout << "Please input next number:";
    }

    cout << "bye." << endl;
    cout << "The sum is " << sum << endl;
    return 0;
}
```
